  • 原理介绍

    • 模糊综合评判模块的主要功能是为战略家提供各种模糊现象的定量处理方法，以便做出较好的决策。本模块的目标是：给定目标，指标集和评语集，使用合适的模糊算子对各因素进行综合评判。用户首先给出目标，指标集，评语集，再根据相应的隶属度矩阵，后台程序计算输出结果。具体理论知识如下：

    设与被评价失误相关的因素有n个，记作U={u1,u2,...,un}，称之为因素集或指标集，考虑用权重A={a1,a2,...,an}来衡量各因素重要程度的大小。又设所有可能出现的评语有m个，记作V={v1,v2,...,vm}，称之为评语集或评判集。

    模糊综合评判的步骤：

    1. 确定因素（指标）集U={u1,u2,...,un}；
    2. 确定评判（评语）集V={v1,v2,...,vm}；
    3. 进行单因素评判得到隶属度向量ri=(ri1,ri2,...,rim)形成隶属度矩阵,如下图7；


      4. 图8. 隶属度矩阵

    4. 确定因素集权重向量，对评判集可数值化或归一化；
    5. 计算综合判断（综合隶属度）向量：对于权重A={a1,a2,...,an}，计算B=A*R（其中“*”为模糊算子）；
    6. 根据隶属度最大原则做出评判，或计算综合评判值

    模糊综合评判法的主要算子有4个

    1. M(∧,∨)算子，∧表示取小，∨表示取大


    2. M(·,∨)算子，·表示相乘


    3. M(∧,+)算子，+表示相加


    4. M(·,+)算子


    以上四个算子在综合评价中的特点如下图8：


    图9. 四个算子的特点

    在本模块中，我们选取第4个算子作为模糊算子进行计算，该算子体现权数作用明显，综合程度强，利用R的信息充分
